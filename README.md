# Indian Calculation Practice

インド式計算（ヴェーダ数学）の掛け算を練習する Python 製CLIアプリです。

暗算力向上を目的として、インド式計算でよく利用される掛け算パターンをランダムな問題形式で練習できます。

## 機能

### 1. 十の位が同じ数同士の掛け算

例：

* 13 × 15
* 24 × 28

### 2. 十の位が同じで一の位の合計が10になる掛け算

例：

* 23 × 27
* 64 × 66

### 3. 一の位が同じで十の位の合計が10になる掛け算

例：

* 64 × 44
* 83 × 23

### 4. ×11 の掛け算

例：

* 11 × 15
* 18 × 11

## 工夫した点

* `random` モジュールを使用して問題をランダム生成
* `try-except` による例外処理を実装
* 不正入力時でもプログラムが停止しない設計
* 全角数字入力に対応
* 全角「ｂ」入力でもメインメニューへ戻れるよう対応
* 問題数をカウントして継続学習しやすくした

## 使用技術

* Python
* random モジュール

## 実行方法

```bash
python VedicMathmatics.py
```
